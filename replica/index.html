Cascading Style Sheet

CSS é chamado de linguagem Cascading Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página. Utilizando o CSS é possivel alterar a cor do texto e o fundo, fonte e espaçamento entre parágrafos. Também pode criar tabelas, usar variações de layoults, ajustar imagens para suas respectivas telas e assim por diante.
