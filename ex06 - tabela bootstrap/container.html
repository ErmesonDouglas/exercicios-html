HTML

(HyperText Markup Language): É a linguagem de marcação usada para estruturar o conteúdo da web. Define elementos como títulos, parágrafos, links e imagens.

CSS

(Cascading Style Sheets): É a linguagem usada para estilizar e layoutizar o conteúdo HTML. Controla cores, fontes, espaçamentos e posicionamento dos elementos.

JAVASCRIPT

JavaScript: É uma linguagem de programação que adiciona interatividade e comportamento dinâmico às páginas web. Permite manipular o conteúdo HTML e CSS em tempo real, responder a eventos e muito mais.
